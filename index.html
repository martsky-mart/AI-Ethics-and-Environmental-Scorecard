How to Host & Deploy This Web App

Option 1: Host Locally (Instant)

This is a pure client-side static HTML file. You can simply double-click the file to open it in any web browser, or launch a quick local server:

python -m http.server 8000

Option 2: Deploy to Free Static Hosting

Since the app runs completely on the front-end, you can deploy it to any modern serverless/static platform for free:

  • GitHub Pages: Upload this single file named as index.html to a repository and enable Pages in settings.
  • Vercel / Netlify: Simply drag and drop the single HTML file directly into their dashboards.
  • Cloudflare Pages: Upload to Cloudflare dashboard or deploy via CLI with a simple static output rule.
